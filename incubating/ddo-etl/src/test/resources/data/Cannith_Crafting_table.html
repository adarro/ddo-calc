Cannith Crafting/table 1b

From DDO wiki
< Cannith Crafting
Jump to: navigation, search

This table lists all prefix, suffix, and extra effects that can be added on to different gear via Cannith Crafting. Note that the "Extra" effect is only available to be applied to items ML 10 and higher.

Note: "Insightful" effects (including Parrying) cannot be applied to items with ML < 10, even in prefix/suffix locations.

  • See also: Compiled listing of all Cannith Crafting Tables
