case class ReqSkill(id: Int, amount: Int, trained: Boolean = false) extends EnumEntry with Requirement with Product with Serializable

Represents the minimum level of the skill to activate an ability or qualify for an enhancement.

id

Identifies the skill, i.e. 'Bluff'

amount

the minimum trained or effective level of the skill.

trained

determines if the skill level. true (i.e. the base skill must be explicitly trained to this level. false the total effective skill must be this level. i.e. can include bonuses from items / ability scores etc.
